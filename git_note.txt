
初始化仓库
git init

将文件提交至暂存区
git add <文件名>

将文件提交至仓库
git commit -m "本次提交说明" "文件名"

查看当前仓库状态
git status

查看历史提交记录
git log   #展示所有版本的详细信息
git log --oneline  #以一行的方式展示版本信息
git reflog    #显示head信息，一般采用这种

回退版本
git reset --hard <版本id，一个16进制数>
hard 参数表示的是，仓库中的代码和暂存区和工作区中的代码都回到指定版本
可用于在删除工作区中的数据时，想恢复时，可直接回退到之前的状态
其他参数
--mixed
本地库的指针移动的同事，重置暂存区，但工作区不动
--soft  
本地库的指正移动的时候，aa，暂存区和工作区都不移动
